## Installation

To set up the project locally, follow these steps:

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/portfolio-website.git
   ```

2. Navigate to the project directory:

   ```bash
   cd portfolio-website
   ```

3. Install the dependencies:

   ```bash
   npm install
   ```

## Usage

1. Run the development server:

   ```bash
   npm start
   ```

   The application will be available at `http://localhost:3000`.

2. Customize the content:

   - Modify the `src/components/Projects.js` file to add your projects.
   - Edit the `src/components/Skills.js` file to update your skills and experiences.
   - Customize other components as needed to suit your preferences.
